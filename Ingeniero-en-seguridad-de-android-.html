Ingeniero en Seguridad de Android

Como Ingeniero de Seguridad, ayudarás a mantener los sistemas informáticos reforzados contra ataques y brindar servicios de seguridad para proteger datos confidenciales de clientes. También trabajarás con Ingenieros de Software para identificar y reparar de forma proactiva fallos de seguridad y vulnerabilidades.

Requisitos

- Licenciatura en Ciencias de la Computación o experiencia práctica equivalente.
- 2 años de experiencia en testeo de vulnerabilidad a nivel de aplicación y auditar seguridad a nivel de código.
- Experiencia en programación (ej. Python, C/C++, Java o Go).
- Experiencia en el desarrollo de aplicaciones Android (p. ej., C++, Java y/o Kotlin). (Deseable)
- Conocimientos de ingeniería de seguridad, seguridad informática y de redes, autenticación, protocolos de seguridad y criptografía aplicada. (Deseable)


Responsabilidades

- Realizar investigaciones para identificar posibles vectores de ataque contra las aplicaciones de Android.
- Trabajar con otros miembros del equipo para mejorar el conocimiento sobre las vulnerabilidades de las aplicaciones y proponga métodos para detectar y mitigar nuevos tipos de fallas.
- Documentar y proporcionar capacitación para analistas operativos sobre cómo identificar tipos particulares de vulnerabilidades de aplicaciones.
